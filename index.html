03 / MCP-экосистема

MCP в рекрутинге

Model Context Protocol становится стандартом для AI-агентов. Job-платформы, у которых нет MCP, будут парситься неофициально (Apify уже парсит HH.ru). Лучше контролировать нарратив.

Хронология MCP-запусков

12 января 2026

Dice.com — первый MCP для вакансий

13+ параметров фильтрации, natural language queries, без авторизации. Получил значительное внимание индустрии.

Январь-февраль 2026

Indeed — комплексный MCP (beta)

Job search + job detail + resume + company data. Полная career intelligence через AI-агентов.

Февраль 2026

СБОРКА Career MCP — первый русский!

5 tools: salary_data, job_market_trends, resume_review, interview_prep, career_advice. PyPI + SSE endpoint. Ноль конкурентов на российском рынке.

Community

JobSpy, LinkedIn MCP, Apify HH Scraper

Неофициальные MCP-серверы: агрегация Indeed+LinkedIn+Glassdoor, автоматизация откликов, парсинг HH.ru.

Стратегическая возможность

Ни одна российская платформа не имеет MCP-сервера. HeadHunter, SuperJob, Rabota.ru, Avito — никто. СБОРКА уже запущена на PyPI и mcp.sborka.work/sse. Расширение до marketplace tools (submit_job, submit_resume, find_matches) делает СБОРКУ единственным AI-native входом на русский рынок труда.

04 / Технологии матчинга

Эволюция алгоритмов

Все российские платформы застряли на Gen 1-2 (keyword matching). State of the art использует semantic embeddings + graph neural networks + LLMs с точностью 91%+.

Поколение Технология Точность Кто использует Ограничения
Gen 1 Keyword matching ~40-50% HH.ru, SuperJob, все RU Пропускает синонимы, контекст
Gen 2 TF-IDF + cosine similarity ~60-70% Indeed, LinkedIn Нет семантики
Gen 3 BERT/SBERT embeddings ~80-85% Eightfold, Mercor Статичные навыки
Gen 4 LLM-based + RAG ~90%+ Moonhub, передовые Стоимость, латентность
СБОРКА BM25 → pgvector → LLM re-rank 80%+ target Фазированный подход $0-2/мес стоимость

Open-source стек СБОРКИ

Не нужен многомиллионный R&D бюджет. Готовые компоненты:

  • PostgreSQL + pgvector — vector similarity search
  • Sentence-BERT (MiniLM) — 384-dim embeddings, работает на CPU
  • DeepSeek V3 — LLM re-ranking за $0.14/1M токенов
  • FastAPI + tsvector — BM25-like full-text search

Преимущество фазированного подхода

Запуск с Phase 1 (keyword), эволюция к Phase 3 (LLM) по мере роста:

  • Phase 1: PostgreSQL FTS — $0/мес, <50ms
  • Phase 2: + pgvector hybrid — ~$2/мес, <100ms
  • Phase 3: + LLM re-ranking — ~$5/мес, <500ms
  • Порог: Phase 2 при 500+ листингов
05 / Pain Points

Боли рынка

Соискатели

Critical

Нет обратной связи

70%+ откликов получают НОЛЬ ответа. Работодатели смотрят резюме и молчат. HH пытается подталкивать, но compliance низкий.

High

Спам и фейковые вакансии

Скам-листинги, MLM под видом работы, bait-and-switch по зарплате. Платформы не фильтруют эффективно.

High

Pay-to-play видимость

Резюме без платного буста скрываются на 10+ страницах. Бесплатный доступ де-факто не работает.

Medium

Нерелевантный matching

Keyword matching показывает нерелевантные вакансии. "Python developer" != "Python programmer" для алгоритма.

Работодатели

Critical

Поток резюме

250 откликов на вакансию, 95%+ нерелевантны. Только 4-6 кандидатов доходят до интервью. Огромные потери времени HR.

High

Высокая стоимость найма

Средняя стоимость найма: $4,700 (США). В России монопольные цены HH.ru постоянно растут.

High

AI-сгенерированные отклики

74% HR-менеджеров обнаруживают AI-контент в резюме. 58% обеспокоены. Новая проблема 2025+.

Medium

Низкая квалификация

93% российских работодателей жалуются на недостаток навыков у соискателей (HH.ru 2025). Проблема matching, не supply.

07 / Возможности

Стратегические возможности

1. MCP-First — первая русская job-платформа

Ноль конкурентов в RU. AI-агенты ищут вакансии через MCP → трафик → конверсии. Стоимость привлечения = $0.

2. Гарантия ответа 72ч

Революция для RU рынка. Deposit от работодателя, возврат только при предоставлении фидбека. #1 pain point solved.

3. Semantic Matching для русского языка

Все RU платформы на keyword matching. SBERT + pgvector = 80%+ релевантность vs 40-50% у HH. Open-source стек.

4. Community Verification

Живые встречи СБОРКИ = верификация кандидатов. "Этот человек прошёл 3 mock-интервью." HH это не может повторить.

5. Ценовой дисраптор

HH.ru с маржой 60% оставляет огромное пространство. Даже при 30% марже СБОРКА предложит в 5-10x дешевле.

6. UGC SEO-маховик

Каждая вакансия и резюме = индексируемая страница. Schema.org JobPosting → Google for Jobs. Бесконечный органический трафик.

Ключевой вывод

СБОРКА занимает уникальный квадрант: Community-driven + AI-native + Russian market. Ни один конкурент не находится здесь. Первый MCP-сервер для русского рынка труда, semantic matching вместо keywords, верификация через живое сообщество. Всё это на инфраструктуре за $5/мес.
