01 / Cold Start

Как гиганты решали chicken-and-egg

Uber

Hard side first (drivers). Cold-called black car services, guaranteed $30/hr. City-by-city с hyperlocal density.

Для СБОРКИ: Seed supply (вакансии) через агрегацию hh.ru + TG каналов

Airbnb

Supply seeding через события. Конференции с sold-out отелями. Cross-posting Craigslist. Фото хостов бесплатно.

Для СБОРКИ: Вебинары + живые встречи = события для привлечения обеих сторон

Tinder

Manufactured demand через exclusivity. Campus-by-campus, sorority-fraternity паринг, FOMO.

Для СБОРКИ: "СБОРКА Verified" badge как эксклюзив для прошедших mock-интервью

Atomic Network СБОРКИ

По Andrew Chen: Atomic Network — минимальная сеть, дающая standalone ценность. Для СБОРКИ: один работодатель с открытыми ролями + достаточно квалифицированных кандидатов в нише.

Преимущество СБОРКИ: обе стороны уже есть

В отличие от чистого стартапа, у СБОРКИ 26 CRM-пользователей (соискатели) и связи с работодателями через карьерный коучинг. Cold start problem реален, но смягчён. Стратегия: Supply-First с синтетической ликвидностью — агрегируем 5,000-15,000 вакансий с HH.ru в день 1, marketplace выглядит полным.

MCP как уникальный канал трафика

MCP Agent Loop (novel)

AI-агент использует СБОРКА MCP → возвращает data + CTA → юзер заходит на sborka.work → регистрируется → говорит AI-ассистенту про СБОРКУ → больше MCP-запросов.

CAC = $0. Ни один конкурент на русском рынке не имеет MCP.

Salary Data Strategy (Glassdoor model)

HH.ru API → компиляция зарплатных данных по роли/городу/грейду → красивые визуализации на sborka.work → viral sharing (люди любят проверять зарплаты) → СМИ цитируют → LLMO authority.

03 / Web3 Future

Web3 потенциал

Вердикт: НЕ реализовывать сейчас. 26 юзеров слишком мало для токен-экономики. Регуляторная неопределённость. Marketplace должен работать в Web2 сначала. Но — проектировать с учётом миграции.

Active

Braintrust

Decentralized talent network. BTRST token. 0% fee for talent.

102K+ talent, Fortune 1000 clients. Pivoted to AI+Web3.

Урок: Web3 talent marketplace работает at scale, но нужны крупные клиенты + AI pivot

Niche

Human Protocol

Decentralized data labeling + work distribution. HMT token.

Smart contract escrow for work. Узкий рынок.

Урок: Escrow работает, но ниша ограничивает рост

Early

TalentLayer

On-chain work history protocol. Cross-platform reputation.

Portable reputation — реальная проблема. Adoption пока медленный.

Урок: Protocol layer перспективнее application layer

TON интеграция (Phase 4)

Что готовить сейчас

  • Reputation system в Web2 (DB) с миграцией на on-chain
  • TON Connect для опционального wallet auth в TMA
  • POAP-style attendance badges для вебинаров

Что делать в Phase 4 (6+ мес)

  • Soulbound Tokens (SBTs) для верификации навыков
  • Smart contract escrow для фриланс-найма
  • TON Jettons для reputation/rewards
  • TON DNS для профилей (user.sborka.ton)
04 / Roadmap

4-фазный план реализации

Phase 1 — MVP (2 недели)

"Concierge Marketplace"

  • Web-форма: Submit Job на sborka.work/post-job
  • Web-форма: Submit Resume
  • Страница sborka.work/jobs (листинг)
  • MCP tools: submit_job, submit_resume, find_matches
  • Seed: 50 вакансий с HH.ru API
  • Manual matching первых 5 пар
  • Schema.org JobPosting на каждой вакансии
Target: 10 вакансий, 50 резюме, 5 matches
Phase 2 — AI Matching (1 месяц)

Semantic + Dashboard

  • LLM semantic matching (Groq free tier)
  • Employer dashboard: кандидаты, scores, контакты
  • Candidate dashboard в TMA: matched jobs, статусы
  • TG bot уведомления: "87% fit для PM at X"
  • Email capture: SEO → popup → weekly digest
  • Referral program v1
  • 10 salary pages с HH.ru данными
Target: 50 вакансий, 200 резюме, 20 employers
Phase 3 — Verification + Premium (2-3 мес)

Монетизация

  • СБОРКА Verified badge (mock-интервью)
  • Premium employer: 5,000 RUB/мес
  • Premium candidate: 1,990 RUB/мес
  • Market analytics dashboard
  • Google for Jobs включение
  • Partnership с 3-5 кадровыми агентствами
  • UGC: 200+ индексируемых страниц
Target: 200 вакансий, 1,000 резюме, 10 paying employers
Phase 4 — Scale + Web3 (6+ мес)

Масштабирование

  • API для HR систем (Huntflow, Talantix)
  • TON Connect wallet auth
  • Reputation SBTs
  • Smart contract escrow
  • Yandex Rabota интеграция
  • English version для CIS/remote
  • DAO governance pilot
Target: 1,000+ вакансий, 5,000+ резюме

Стратегический вывод

СБОРКА может стать первой AI-native двусторонней платформой для русского рынка труда. Уникальная комбинация: MCP-вход для машин (ноль конкурентов), semantic matching (vs keyword у HH), community verification (живые встречи), и near-zero infrastructure cost ($5/мес). Break-even при 5 работодателях. Potential: до 3.5M RUB/мес MRR за 12 месяцев (aggressive). Phase 1 MVP — 2 недели.
